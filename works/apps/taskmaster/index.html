App: TaskMaster

This Portfolio is an individual activity for the subject Introduction to Computing.


When: 2023

Method: Appsheet

Role: Author


This is my final output for the subject. The task was to make an app using Google Appsheet. We were told to choose and use a template from appsheet and just edit the contents. But I didn't liked the template for the to do list so I made my own. I created my own database using Google Sheets and modified the UI for better user experience. The app has different views. First is the Task view which contains the user's tasks, next is the Pending view which contains the user's pending tasks, next is the Task Calendar which shows the unfinished and not yet started tasks to inform the user and finally, the About page of the app which contains information about the app- where I sorta trolled by adding Anakin Skywalker's quote about sand. The app also uses bot to send an email to the user once a task's due date is near.
